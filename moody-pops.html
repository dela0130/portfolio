About Moody Pops

Simple start

For the final project in the first semester, Kev D was required to create a website for a hypothetical customer who was a cake pop baker. The design thinking process was simulated based on a brief description of the entrepreneur's needs. This website design was a part of the final submission. Once submitted, Moody Pops was put to bed.

During his second semester midterm project, another assignment forced the resurrection of Moody Pops. The midterm project required him to code a website to match the design using HTML and CSS. The trick was to also make it responsive and interactive. You can see the code here on Kev. D's GitHub account. Check out his code, he left comments. Now Moody Pops is immortal!

Real life?

Could a cake-pop-only bakery actually work? Kev D. estimated the viability to stay inspired. A shop would need to sell 5000 cake pops per month, on average. How many would you buy?
